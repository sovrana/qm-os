# Improvement Suggestions

Self-improving system: suggestions extracted from conversations, tracked by frequency. When a suggestion appears 3+ times, it becomes a **BEAD** (ready to implement).

**How it works:**
1. `/transform session` extracts suggestions from each substantive conversation
2. New suggestions are fuzzy-matched against existing ones (Claude does semantic matching)
3. Similar suggestions increment the count; truly new ones are added
4. Beads (3+ occurrences) surface in `/weekly` for implementation
5. Implemented suggestions move to the archive section
