## Why This Exists

Claude Code is the most capable AI coding tool available. It's also stateless. Every conversation starts from zero. No memory of what you decided yesterday, no awareness of how your stakeholders prefer communication, no accumulation of the corrections you've already made.

I needed an AI assistant that compounds. One where every session makes the next one better. Where a rejected framing becomes a permanent constraint. Where a cross-theme insight gets captured, not forgotten.

Nothing like this existed. So I built it.

## The Timeline

8 weeks. Daily production use across multiple active workstreams. Not a weekend project or proof of concept.

Week 1-2: Core vault structure, task management, first skills (/morning, /inbox). Week 3-4: Memory system, semantic search, hooks automation. Week 5-6: Advanced skills (/challenge, /weekly, /draft), self-improvement loop. Week 7-8: Calibration system, rules architecture, voice matching.

The CLAUDE.md file now contains dozens of rules that were discovered through use, not designed upfront. That's the point.

## The Thesis

Knowledge workers spend most of their time on coordination - filing, retrieving, prioritising, remembering context, adapting communication style per audience. AI agents can handle all of this, if you architect the system.

Quartermaster is that architecture. Open source. MIT licensed. Built to be forked and adapted.
